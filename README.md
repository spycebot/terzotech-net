## Summary


### Objectives
1. Flask development environment on Cloud Linux
2. Cloud based TTS

### Dependencies

- python3-pip
- python3-dev
- build-essential
- libssl-dev
- libffi-dev
- python3-setuptools

+ python3-venv

$ pip install wheel
